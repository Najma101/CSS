CSS Colors – Full Guide

CSS offers many ways to set color: names, RGB, HEX, HSL, and opacity (RGBA/HSLA).

1. CSS Color Names

CSS has ~140 predefined color names like red, blue, gold, tomato, etc.

This box uses a **color name** (orchid).

2. RGB (Red, Green, Blue)

RGB defines color using red, green, and blue values from 0 to 255.
Example: rgb(255, 0, 0) = red

This box uses **RGB**: rgb(100, 180, 255)

3. HEX (Hexadecimal)

HEX is a 6-digit code representing colors using numbers + letters (0-9, A-F).
Example: #ff0000 = red

This box uses **HEX**: #6a0dad

4. HSL (Hue, Saturation, Lightness)

HSL stands for:

  • Hue: 0–360 (the color on the color wheel)
  • Saturation: color intensity (0%–100%)
  • Lightness: brightness (0%–100%)

Example: hsl(200, 80%, 60%)

This box uses **HSL**: hsl(200, 80%, 60%)

5. Opacity (Alpha Channel)

Opacity controls transparency.

Three main ways:

  • opacity property → affects whole element
  • rgba() → RGB + alpha (0 to 1)
  • hsla() → HSL + alpha (0 to 1)

opacity: 0.5 (affects whole box)
rgba(255, 99, 71, 0.5) — semi-transparent tomato
hsla(200, 80%, 60%, 0.5) — semi-transparent blue
